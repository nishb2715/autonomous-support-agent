# 🚀 Autonomous Customer Support Triage & Resolution Agent

An end-to-end agentic AI system that automatically triages, resolves, and escalates customer support tickets using machine learning, retrieval-augmented generation (RAG), decision logic, and adaptive feedback learning.

---

## 📌 Project Overview

This system simulates an enterprise-grade autonomous support agent capable of:

1. Reading incoming customer support tickets
2. Classifying intent and urgency
3. Retrieving relevant company knowledge
4. Deciding whether to:
   - Auto-resolve
   - Ask clarifying questions
   - Escalate to a human
5. Generating structured responses
6. Learning from human feedback signals
